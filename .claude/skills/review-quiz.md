# 随机抽题复习

从用户已积累的面试题中随机抽取题目，检验记忆和理解程度。

## 触发

用户使用 `/review-quiz` 或要求抽题复习时触发。

## 流程

1. 读取 `C:\Users\76741\Desktop\面试\面试题.md`
2. 从中随机选取 1 道题目，只展示题目，不展示答案
3. 等用户作答
4. 用户回答后，对比文件中的参考答案，给出：
   - 回答完整度评估
   - 遗漏的关键点
   - 补充说明
5. 询问是否继续下一题

## 抽题原则

- 如果用户指定方向（如 MySQL、并发、JVM），则从对应分类中抽取
- 否则随机跨分类抽取
- 优先抽取用户之前回答不好或没覆盖到的题目
